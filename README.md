# Tic-Tac-Toe Game

**Live Demo:** https://pateru-pavan-kumar.github.io/Tic-Tac-Toe-Game/

## Overview
A simple, responsive Tic-Tac-Toe game built with **HTML**, **CSS**, and **JavaScript**. Ideal as a beginner project demonstrating DOM manipulation, game logic (win/draw detection), and basic UI.

## Features
- Two-player local gameplay (X vs O)  
- Win detection (rows, columns, diagonals)  
- Draw detection  
- Reset / New Game buttons  
- Responsive, minimal UI

## Tech Stack
- HTML5 (`index.html`)  
- CSS3 (`style.css`)  
- JavaScript (`app.js`)
